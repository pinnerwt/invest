# Verdict: 1907 永豐餘 (2026-05-11)

**Verdict: 不買** (confidence: medium)
**Top reasons:**
1. 大賺年 (2020/2021 EPS ~3.14) 配息率僅 ~30-48%，**未達 robertshih 50% 門檻** → 6x P/E 規則不嚴格適用
2. 集團控股結構 (造紙 + 紙器 + 化學 + 元太轉投資 + 永豐金) 使其 borderline cyclical；現價 $26 已接近近年高位，且 EPS 走下坡 (3.14 → 1.19)

---

## Stage 1 — Regime: cyclical (borderline — 集團控股稀釋)

### EPS 歷史 (2015-2025)
| 年 | EPS | 現金股利 (隔年配發) |
|---|---|---|
| 2015 | 0.30 | 0.3 |
| 2016 | 0.08 | 0.9 |
| 2017 | 0.97 | 0.6 |
| 2018 | 0.87 | 0.6 |
| 2019 | 1.38 | 0.7 |
| 2020 | 3.14 | 1.5 |
| 2021 | 3.13 | 1.5 |
| 2022 | 1.31 | 0.9 |
| 2023 | 1.10 | 0.9 |
| 2024 | 0.98 | 0.8 |
| 2025 | 1.19 (Q3 trailing) | — |

**EPS 振幅 = 3.14 / 0.08 = 39x** → cyclical 確認 (但 0.08 為異常低值，較合理區間 0.30-3.14 仍有 10x，明確 cyclical)

**Borderline 註記**: 永豐餘是控股公司，造紙本業 + 紙器包裝 + 元太 (10.56%) + 永奕 + 永豐金等多元化，EPS 受非紙業因素影響。但主力營收仍為造紙 (工紙 + 文化用紙 + 家庭用紙)。

**判定: cyclical** → proceed to Stage 2

## Stage 2 — Payout gate: ⚠️ borderline fail

10 年 EPS 中位數約 1.10，× 1.5 = 1.65 → 大賺年為 2020 (3.14)、2021 (3.13)、2019 (1.38 borderline)

**重要: 台股配息規則「隔年配發」**
| 大賺年 | 該年 EPS | 次年現金股利 | 配息率 |
|---|---|---|---|
| 2020 | 3.14 | 2021 配 1.50 | **47.8%** ❌ |
| 2021 | 3.13 | 2022 配 1.50 | **47.9%** ❌ |
| 2019 (borderline) | 1.38 | 2020 配 0.70 | 50.7% ✅ |

**兩個明確大賺年配息率都未達 50%**。改包含「現金 + 股票股利」也未上修 (永豐餘近 10 年無股票股利)。

**Gate: FAIL** — robertshih 6x P/E floor rule 不嚴格適用。

近 5 年平均現金股利 1.12 元，殖利率 3.6% — 屬於配息穩定但 payout 偏保守的集團型公司。

## Stage 4-5 — EPS 三情境 + Floor (informational)

| 情境 | 假設 | EPS |
|---|---|---|
| Bear | 紙漿成本續漲 + 文紙需求疲弱 | ~0.8 |
| Base | 持平於現況 | ~1.2 |
| Bull | 文紙漲價落實 + 工紙循環翻揚 | ~2.0 |

**Floor (如適用) = 0.8 × 6 = $4.8**

現價 $26.05，gap = (4.8 - 26.05) / 26.05 = **-82%** → 機械上嚴重高估

但因 Stage 2 fail，floor rule 不該直接套用 — 現價合理性應改以「集團 NAV + 持股部位市值 (元太、永豐金)」評估，不在 framework 範圍。

## Stage 8 — Verdict reasoning

Stage 2 payout gate fail 已使 6x rule 不嚴格適用，且 checklist 僅 1-2/6。Priority 2「不買」條件 (gap ≤ 0 + checklist ≤ 3) 滿足。
