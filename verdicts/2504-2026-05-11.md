# Verdict: 2504 國產 (2026-05-11)

**Verdict: 不適用** (defensive hybrid — 預拌混凝土寡占 + 不動產 + 加油站) (confidence: medium-high)

**Top reasons:**
1. 預拌混凝土市佔 ~16-18% (與台泥 / 亞泥下游)，台灣寡占且地理門檻高 (運距 < 90 分鐘) — EPS 振幅 < 2x，遠低於 5x cyclical 門檻；EPS 軌跡 3.3 (2023) → 4.13 (2024) → 推升軌跡
2. 業務組合多元 (預拌混凝土 90% + 防火建材 5% + 不動產買賣 / 出租 + 加油站) — 不動產資產活化 + 加油站通路平滑營建 cycle beta，屬 defensive hybrid

## Stage 1 — Regime: defensive (hybrid 預拌混凝土 + asset_play tilt)

業務組合：
- 預拌混凝土 (~90%): 台灣最大廠 市佔 16-18%，地理寡占 (運距 90 分鐘限制)
- 防火建材 (~5%): 利基產品
- 不動產買賣與出租: 資產活化 (歷史土地)
- 加油站通路 + 砂石採掘 / 碎洗 / 運銷

EPS 軌跡：
- 2023: 3.30 元 (擬配息 2.6 元 = payout 79%)
- 2024: 4.13 元 (預估)
- 殖利率高 (uc913 標 4.5/5 「殖利率稱霸 價格公道合理」)

振幅 4.13 / 3.30 = 1.25x，遠 < 5x cyclical 門檻。即使加上 2020-2022 的低點 (估 2-2.5 元), 振幅 ~2x 上限.

雖然下游連動建設業 (1808 潤隆 / 2542 興富發 等已歸 cyclical 不買), **國產上游關鍵差異**：
1. 預拌混凝土地理寡占 → 不像 cement 受全球 commodity 衝擊
2. 不動產 / 加油站業務 buffer
3. 配息穩定 (近年 2.6+) 顯示 stable cash flow

歸類 defensive hybrid (與 2107 厚生 橡膠+地產 hybrid 防禦混業 同邏輯，cache 內 2107 已歸 defensive)。

**Asset_play hybrid tilt 觀察：** 公司「積極運用其豐厚的土地資產，進行不動產買賣與出租業務」— 屬 asset utilization 而非 asset_play (本業未停業 + 配息正常)。可備觀察未來若預拌混凝土萎縮、不動產出售獲利主導 EPS，可再評估 asset_play.

## Stage 2-9 — 不執行 (Stage 1 fail-fast)
