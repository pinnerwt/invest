# Verdict: 1722 台肥 (2026-05-11)

**Verdict: 不適用** (confidence: high)
**Top reasons:**
1. EPS 8 年振幅 4.00 / 1.04 = **3.85x**，未達 cyclical 5x 門檻。
2. 業務結構已從化肥本業（虧損）轉向：(a) 中東朱拜爾肥料 (JFC) 轉投資收益佔大頭、(b) 南港經貿園區 / 高雄海洋科技園區 租金收益、(c) 化工事業獲利支撐 — 本質為 asset_play / hybrid defensive，非純尿素 cyclical。

---

## Stage 1 — Regime

**年度 EPS（元）:**
| 年 | EPS |
|---|---|
| 2018 | 2.33 |
| 2019 | 2.11 |
| 2020 | 2.50 |
| 2021 | 3.10 |
| 2022 | 2.72 |
| 2023 | 4.00 |
| 2024 | 2.01 |
| 2025 | 1.04 |

- 振幅: 4.00 / 1.04 = **3.85x**（未達 5x cyclical 門檻）
- 業務結構（依 2025 法說會公開資訊）:
  - 肥料本業: 2025 前 3 季虧損 ~11 億（政府凍漲補貼模式，肥料價格政策限制）
  - 化工事業: 2025 前 3 季獲利 ~12 億
  - **中東朱拜爾肥料 (JFC) 認列投資收益**: 2025 前 3 季 9.71 億（最大獲利變動因子）
  - 不動產租金 (南港經貿園區 + 高雄海洋科技園區): 穩定現金流，本質 REIT-like
- 連 25 年發放現金股利，5 年平均殖利率 4.02%
- caller 預期 indicator = urea 國際 spot price，理論上應驅動 EPS，但實際:
  - 國內肥料價格凍漲 → 即使 urea 國際價飆升，台肥內銷不受惠
  - 中東 JFC 受惠 urea 價（但只是合資 33% 分潤）
  - 不動產 + 化工部位完全與 urea 脫鉤
- → **indicator → EPS 傳導被結構性稀釋**

**Regime: asset_play / hybrid_defensive**（同 cache 中 1101 台泥 / 1907 永豐餘 邏輯：cyclical 元素被多元化稀釋為 defensive_now）

## Verdict 結論

**不適用**: 
- EPS 振幅 3.85x < 5x cyclical 門檻
- urea 國際價格不再是台肥 EPS 的有效 leading indicator（凍漲政策 + 多元化收入結構）
- 屬 asset_play / hybrid_defensive，類似 1101 / 1907 處理
- 真正的尿素 cyclical 暴險在中東朱肥 JFC（合資公司未上市）
